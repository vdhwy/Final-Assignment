# Final_Assignment

To run the app, create a .env file and paste all the keys in firebase project.

For example: 

EXPO_PUBLIC_API_KEY = your-api-key

EXPO_PUBLIC_AUTH_DOMAIN = your-auth-domain-key

EXPO_PUBLIC_PROJECT_ID = your-project-id

EXPO_PUBLIC_STORAGE_BUCKET = your-storage-bucket-key

EXPO_PUBLIC_MSG_SENDER_ID = your-message-sender-id

EXPO_PUBLIC_APP_ID = your-app-id
